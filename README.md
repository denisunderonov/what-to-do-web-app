# What To Do Web App

Простое веб-приложение, которое предлагает случайные идеи для занятий, когда вам скучно.

## Возможности

- Получение случайных идей через Bored API
- Три варианта на выбор
- Минималистичный дизайн

## Технологии

- HTML5
- CSS3
- JavaScript (Fetch API)
- Bored API
